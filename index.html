MLB Award Winner Visualization

About this project:

This project visualizes award winners in Major League Baseball from 2000 to 2023. Using interactive filters, you can explore MVP Winners, Cy Young Winners, and Silver Slugger Winners over any given season. There are various graphs for each award that showcases the winner's stats from their award-winning year.

Data Source:

This project uses data from the Lahman Baseball (Version 2023), a publicly available dataset containing MLB player statistics from 1871-2023.

Skills:

  • SQL - Stored cleaned data in SQLite db.
  • Python - Data cleaning and processing.
  • Tableau - Data visualization and dashboard creation.
